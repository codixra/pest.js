<Note type="success">
  PEST.js is a structured boilerplate for building Node.js applications with Express.js and TypeScript.
</Note>

## Quick Start

<Stepper>
  <StepperItem title="Get the Template">
    ```bash
    # Clone the repository
    git clone https://github.com/username/pestjs-template my-app
    cd my-app
    ```
  </StepperItem>

  <StepperItem title="Install Dependencies">
    ```bash
    npm install
    ```
  </StepperItem>

  <StepperItem title="Configure Environment">
    ```bash
    cp .env.example .env
    # Update the .env file with your settings
    ```
  </StepperItem>

  <StepperItem title="Start Development">
    ```bash
    npm run dev
    ```
  </StepperItem>
</Stepper>

## Features

<Note type="note">
Core features included in this template:

- Feature-based folder structure
- TypeScript configuration
- MongoDB integration with Mongoose
- Basic security middleware
- Error handling and logging
- Testing setup with Jest
- Docker configuration
</Note>

## Key Features

<Note type="note">
PEST.js comes with several built-in features to accelerate your development:

- **TypeScript Support**: Built-in TypeScript configuration
- **Security Features**: CORS, Helmet, Rate Limiting
- **Database Integration**: MongoDB with Mongoose
- **API Documentation**: Automated Swagger/OpenAPI docs
- **Testing Setup**: Jest and Supertest configuration
- **Docker Support**: Ready for containerization
</Note>
