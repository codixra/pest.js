<Note type="note">
  PEST.js generates Node.js projects through a simple CLI interface and modular file generators.
</Note>

## Framework Execution

<Stepper>
  <StepperItem title="CLI Entry">
    Framework starts with:
    ```bash
    ./pestjs
    ```
    Validates environment and collects user input
  </StepperItem>

  <StepperItem title="User Input">
    Prompts for project details:
    - Project name (default: pestjs-app)
    - GitHub username (default: your-username)
  </StepperItem>

  <StepperItem title="File Generation">
    Creates project files:
    - package.json with dependencies
    - tsconfig.json for TypeScript
    - .eslintrc.json for linting
    - src/app.ts for Express app
    - .env files for environment
  </StepperItem>

  <StepperItem title="Git Setup">
    Initializes repository:
    ```bash
    git init
    git add .
    git commit -m "Initial commit: PEST.js project"
    ```
  </StepperItem>
</Stepper>

## Generated Files

### package.json
```json
{
  "name": "your-project",
  "scripts": {
    "start": "node dist/app.js",
    "dev": "nodemon --exec ts-node src/app.ts",
    "build": "tsc",
    "test": "jest",
    "lint": "eslint . --ext .ts"
  },
  "dependencies": {
    "express": "^4.18.2",
    "cors": "^2.8.5",
    "helmet": "^7.1.0",
    "dotenv": "^16.4.1"
  }
}
```

### src/app.ts
```typescript
import express from 'express';
import cors from 'cors';
import helmet from 'helmet';

const app = express();

app.use(helmet());
app.use(cors());
app.use(express.json());

app.get('/', (_, res) => {
  res.json({
    message: 'Welcome to your-project API',
    version: '1.0.0'
  });
});

const PORT = process.env.PORT || 3000;
app.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});
```
